# Frontend дипломного проекта Movies

Сервис, в котором можно найти фильмы по запросу и сохранить в личном кабинете.

## Используемые технологии

* HTML
* CSS
* BEM 
* Flexbox
* Grid Layout
* Media Queries
* React

## Запуск проекта

`npm run start`
